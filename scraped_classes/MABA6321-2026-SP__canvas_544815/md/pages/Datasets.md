## Datasets

| For installing MySQL Server or MySQL workbench, please see [syllabus > Installation Instructions](https://canvas.umn.edu/courses/544815/assignments/syllabus "Syllabus").

The following are datasets that we use in our lectures, labs, and homework assignments. Each dataset has three files

* **[name].sql (Table Creation Script)**: contains SQL scripts that will **create and populate** tables in the currently active database
* **[name]-remove.sql (Table Removal Script)**: contains SQL scripts that will **remove** the tables in the currently active database.
* **[name].png**: **ER Diagram** for the database.

**To run the .sql file using MySQL workbench (see demo video at the end)**,

* **File** > O**pen SQL Script**
* **Select all** (Ctrl +A)
* **Query** > **Execute (All or Selection)** -- The **bolt** icon.
