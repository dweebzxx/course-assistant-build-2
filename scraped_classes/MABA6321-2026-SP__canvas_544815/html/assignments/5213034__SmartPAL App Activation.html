Step 1: Mobile App Installation (or Update)

Search and install "SmartPAL" in App Store or Google Playstore. If you've downloaded the app before, please update it to the latest version. The app icon is as follows:

Step 2: Account Sign Up

If this is the first time you are installing the app, you will need to sign up for an account.

if you want to sign up using email:

  1. Please click on the Sign Up link in the app
  2. Use your University email (e.g., abcd1234@umn.edu) to sign up
  3. Check your email for the Confirmation code.
  4. Log in the app using your email, password, and confirmation code

if you want to sign up using SSO (Single Sign On):

This option is not available currently; we are working on it and will enable it as soon as possible.
  1. Please click on the SSO (Single Sign On) link in the app, then log in using your university credential.

Step 3: Course Configuration

  1. At the landing page, select the course.
  2. Set a learning goal for this course.*
  3. (On the Mobile app only) Enable Notification for the app.

*Note: If you are enrolled in multiple courses, please repeat Step 3 for each course.
