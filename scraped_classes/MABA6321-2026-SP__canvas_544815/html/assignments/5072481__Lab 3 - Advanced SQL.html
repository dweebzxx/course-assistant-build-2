Lab3-AdvSQL.sql

This lab provides SQL practice problems corresponding to Module 3: Advanced SQL. I have organized all the practice questions into four sections, each of corresponds to a lecture video in this module. Feel free to follow long the demonstration videos when you complete the lab.

Databases used in this lab: invoices, department

Lab completion and submission instructions:

  1. Download Lab3-AdvSQL.sql
  2. Open Lab3-AdvSQL.sql in MySQL workbench.
  3. Complete the SQL queries, and copy/paste the results
    • By default, for all select statements, copy only the first 10 rows.
    • You may also want to copy the number of rows returned as reported in the query response.
  4. After you complete the lab, copy and paste it to a Google Doc.
  5. Then, select all the lab code, Insert > Building Blocks > SQL. This will format the Lab codes properly.
  6. Download the google doc as a PDF file: File > Download > PDF Document
  7. Submit the PDF file to Canvas.
